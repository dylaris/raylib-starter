# raylib-starter

A template to start 'raylib' project on **windows** and **linux**.

## Usage

You might be missing graphics dependencies such as GLFW. Follow the compilation prompts to install them.

```console
$ gcc -o nob nob.c
$ ./nob
$ ./main
```
